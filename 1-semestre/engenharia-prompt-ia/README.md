# 🤖 Engenharia de Prompt e Aplicações em IA

Esta pasta é dedicada ao estudo e armazenamento de técnicas práticas de **Engenharia de Prompt** e integrações com **Modelos de Linguagem de Grande Porte (LLMs)** desenvolvidas no primeiro semestre.

## 📌 Conteúdos Principais

* **Técnicas de Prompting:**
  - *Few-Shot Prompting* (exemplos práticos no prompt).
  - *Chain-of-Thought* (orientar a IA a pensar passo a passo).
  - *Role-Playing* (definir personas de altíssima performance para a IA).
* **Estruturação de Outputs:** Prompts focados em gerar saídas limpas em JSON, Markdown ou código estruturado.
* **Automações e Scripts:** Pequenos códigos para automação de tarefas cotidianas usando IAs generativas.

## 🛠️ Ferramentas Estudadas

- OpenAI GPT-4o / ChatGPT
- Anthropic Claude 3.5 / Claude
- Prompt Engineering para Desenvolvimento de Código
