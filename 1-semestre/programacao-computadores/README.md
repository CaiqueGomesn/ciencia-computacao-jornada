# 💻 Programação de Computadores

Esta pasta contém os códigos, exercícios e resoluções de problemas que desenvolvi ao longo da disciplina de **Programação de Computadores**.

## 📌 Conteúdos Principais

* **Lógica e Estruturas de Controle:** Condicionais (`if/else`), laços de repetição (`for/while`).
* **Estrutura de Dados:** Listas, arrays, dicionários e manipulação de arquivos.
* **Algoritmos e Funções:** Modularização de código, funções com parâmetros e tratamento de erros.

## 🛠️ Tecnologias Utilizadas

> [!NOTE]
> Adicione aqui a linguagem padrão usada no seu semestre (por exemplo, **Python**, **C**, **C++** ou **JavaScript**).

## 🚀 Como Executar os Códigos

1. Certifique-se de ter o ambiente da linguagem configurado no seu computador.
2. Abra o terminal na pasta do arquivo correspondente.
3. Execute o comando apropriado (ex: `python nome_do_arquivo.py` ou compilação com `gcc`).
