# 🎨 Interface e Jornada do Usuário

Documentação de pesquisas, mapeamentos de fluxos de telas e estudos sobre a experiência e jornada do usuário (UX/UI) desenvolvidos na disciplina.

## 📌 Conteúdos Principais

* **Mapeamento de Jornadas (User Journey):** Estruturação do passo a passo que o usuário faz para atingir seus objetivos no sistema.
* **Personas:** Criação de perfis fictícios baseados em dados reais para representar as necessidades dos usuários.
* **Arquitetura de Informação:** Definição de menus, fluxos e hierarquia visual das informações.
* **Testes de Usabilidade e Análises Heurísticas:** Avaliação de problemas de usabilidade baseados nas 10 Heurísticas de Nielsen.
