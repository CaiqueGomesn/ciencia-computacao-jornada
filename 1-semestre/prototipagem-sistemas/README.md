# ⚙️ Prototipagem de Sistemas Computacionais

Armazenamento de mockups, wireframes e protótipos interativos criados durante o semestre.

## 📌 Conteúdo Principal

* **Wireframes de Baixa Fidelidade:** Esboços iniciais e fluxogramas de ideias.
* **Protótipos de Média Fidelidade:** Organização estrutural em preto e branco com foco em usabilidade.
* **Protótipos de Alta Fidelidade (Figma):** Interfaces prontas, com design visual de alta performance, tipografia, cores consistentes e interações de transição de tela.

## 🏆 Projeto Destaque: SGPD

O principal projeto desenvolvido foi o **SGPD — Sistema de Gestão de Processos**, um sistema desenhado no Figma focado em agilizar e monitorar fluxos corporativos.
